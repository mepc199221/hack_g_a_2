# hack_g_a_2

Repositorio con dos scripts Python de ejemplo.

## Archivos principales

- `app.py`: imprime dos veces el texto "Segundo hack".
- `app2.py`: imprime el texto "Colaboracion".

## Uso

Ejecuta cualquiera de los scripts con Python 3:

```bash
python3 app.py
python3 app2.py
```

## Objetivo

Este repositorio sirve como demostración sencilla de ejecución de scripts Python y colaboración en el código.

## Notas

Puedes modificar los archivos para añadir lógica adicional o experimentar con nuevos mensajes.
